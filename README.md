# roam-research-awesome-themes

提供两款自用的 Roam Research 自定义主题：

- `light.css`: 亮色主题
- `dart.css`: 暗色主题

把源码复制进 `roam/css` 页面中 css 代码块中即可使用。

Tips: 多主题可以使用 Roam Research 的 Block Version 能力来承载不同版本的 css 代码块即可随时切换主题。
